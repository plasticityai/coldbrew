Coldbrew: Run Python in JavaScript

MODULE_NAME is installed on this page through MODULE_NAME_LOWER.asm.js.
Open the browser console and load Coldbrew with:

MODULE_NAME.load(function() { console.log("Finished loading MODULE_NAME!") });

Once loaded, you can interact with Python. For example, to print the version of Python run the following in the browser console:

MODULE_NAME.run("import sys");
MODULE_NAME.run("print('The current Python version is:', sys.version)");

You can also issue multiple commands in the same function call, to see this, run the following in the browser console:

MODULE_NAME.run(["x = [i for i in range(10)]","print(x)"].join("\n"));

See more documentation on GitHub on interacting with the virtual filesystem, installing modules, interacting with the environment, running Python asynchronously, communicating bi-directionally between the two languages, building a custom Coldbrew Python environment, and more.
